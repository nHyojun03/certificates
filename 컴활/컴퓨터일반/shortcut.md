# Windows 10 단축키 모음

## General

- `ctrl` + `A`: 모두 선택
- `ctrl` + `C`: 복사
- `ctrl` + `V`: 붙여넣기
- `ctrl` + `X`: 선택된 것 잘라내기
- `ctrl` + `Y`: 돌아가기 한 것 다시 실행
- `ctrl` + `Z`: 돌아가기

## 시스템 관련

- `Win` + `I`: 설정 열기
- `Win` + `L`: PC 잠금

## Windows 관련

- `ctrl` + `esc`: 시작 메뉴 표시/닫기

## 유틸리티 관련

- `ctrl` + `shift` + `esc`: 작업관리자

## 파일탐색기 (File Explorer)

- `Win` + `E`: 파일 관리자 열기
- **구성 관련**
  - `Alt` + `D`: 주소 표시줄 선택
  - `ctrl` + `E`: 검색 상자 선택
- **창 관련**
  - `ctrl` + `N`: 새 창 열기
  - `ctrl` + `W`: 활성화된 창 닫기
- **폴더/파일 관련**
  - `ctrl` + `shift` + `N`: 새 폴더
